
# Backend
/lang-portal/backend-flask

## Install
```sh
pip install -r requirements.txt
```

## Setup DB
Database will automatically initialize when running app.py if words.db does not exist
```sh
invoke init-db
```

## Run Backend server
```sh
python app.py
```


# Frontend
/lang-portal/frontend-react

## Install
```sh
npm install
```

## Run Frontend server
```sh
npm run dev
```
